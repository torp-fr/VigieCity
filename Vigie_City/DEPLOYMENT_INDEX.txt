================================================================================
WHAT EACH FILE DOES
================================================================================

DATABASE MIGRATION (20260626000001_freemium_moderation.sql):
  Creates/updates database schema with:
  - moderation_queue: Super-admin review queue (new table)
  - report_flags: Citizen voting/flagging (new table)
  - reports: Adds 4 moderation columns (existing table)
  - municipalities: Adds 4 subscription columns (existing table)
  
  Plus:
  - 7 RLS policies (access control)
  - 2 triggers (auto-count flags, auto-escalate)
  - 5 indexes (performance optimization)

FREEMIUM-AUTO-FILTER (freemium-auto-filter/index.ts):
  Scores report content for spam/profanity:
  - Spam detection: Viagra, casino, URLs
  - Caps detection: ALL CAPS content
  - Punctuation: Excessive !!!, ???
  - Profanity: French words (small list)
  
  If score > 0.7:
  - Sets visible_to_public = false
  - Sets status = pending_review
  - Adds to moderation_queue for admin review
  
  If score <= 0.7:
  - Sets visible_to_public = true
  - Sets status = public
  - Report is immediately visible to citizens

CITY-FETCH-REPORTS (city-fetch-reports/index.ts):
  Fetches reports for city admin dashboard:
  - Requires JWT authorization
  - Filters by municipality where user is admin
  - Only works if subscription_status = 'active'
  
  Returns:
  - List of pending reports (need response)
  - List of resolved reports (with city response)
  - List of escalated reports (flagged by citizens >= 3)
  
  Used by Phase 2 city admin dashboard (not yet built)
