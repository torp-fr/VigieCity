================================================================================
THE 5-STEP WORKFLOW (QUICK OVERVIEW)
================================================================================

Step 1: TERRITORY SELECTION
  Choose "Commune" or "EPCI"
  Search and select from available territories
  Load commune count and population data

Step 2: ADMIN CONTACT
  Enter email, name, password
  Real-time validation (8+ chars, uppercase, digit, special char)
  Shows password strength checklist

Step 3: CONFIGURATION
  Commune Path: Select pricing plan (Hameau/Village/Bourg/Métropole)
  EPCI Path: Table of communes with admin email/name/phone fields
             Add/remove rows dynamically

Step 4: PAYMENT DETAILS
  Select payment date (default: today)
  Choose payment type (Chorus Pro / Virement / Devis)
  Check validation checkbox

Step 5: CONFIRMATION
  Review all entered data
  Click "Valider & créer" to submit
  See results: creation count, failures (if any), created IDs

================================================================================
DATA FLOWS
================================================================================

COMMUNE PATH
  Step 1-4: Gather data into formData object
  Step 5:   Call create-commune Edge Function
  Result:   1 admin created, 1 license created, 1 email sent

EPCI PATH
  Step 1-4: Gather data (EPCI + N commune admins) into formData
  Step 5:   Call create-commune-batch Edge Function
  Result:   1 EPCI admin + N commune admins created
            1 EPCI license + N commune licenses created
            N+1 emails sent
            Detailed results with success/failure counts
