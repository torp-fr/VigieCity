## PROJECT OVERVIEW

**VigieCity** — B2B SaaS application citoyenne for French municipalities.

**Users:**
- Citizens (habitants) : report municipal issues, see local alerts, access civic info
- City admins (agents) : moderate reports, send alerts, manage events/news
- Super-admin (TORP internal) : oversee platform, manage municipalities, handle escalations

**Core Features:**
1. **Alerts & Urgencies** — push notifications for municipal emergencies (water cuts, security, weather)
2. **Citizen Reports** — geolocated problem reports (potholes, vandalism, etc.) with photo/location
3. **Civic Agenda** — public events & municipal activities
4. **Local News** — official municipal updates & deliberations
5. **Citizen Messaging** — direct channel between residents and city services
6. **Freemium Moderation** — auto-filter spam → super-admin review → city admin moderation (when subscribed)
7. **Subscription Tiers** — 6 pricing tiers (Hameau 19€ → Métropole 590€/month) based on population

**Target Devices:**
- iOS app (via Progressive Web App / Capacitor)
- Android app (via Capacitor)
- Web (responsive, desktop + tablet)

## CONTEXT YOU NEED TO KNOW

### Features & Screens
**Citizen-facing:**
- Home: alerts (red banner), quick actions (4 cards), local feed
- Report: form to create issue (category, photo, location, description)
- Map: geolocated reports layer
- Agenda: events with calendar integration
- News: local news feed (official municipal updates)
- Messaging: direct messages with city services

**City admin-facing:**
- Dashboard: KPIs (active alerts, pending reports, citizen engagement)
- Reports queue: pending responses, status tracker, escalations
- Moderation: flagged reports (from freemium auto-filter), approve/reject/escalate
- Settings: tier info, subscription status, user management

**Super-admin (internal):**
- Moderation queue: all flagged reports across municipalities
- City management: add/remove cities, subscription status, billing
- Analytics: municipality KPIs, feature adoption

### Pricing Tiers (6 total)
- **Hameau** (< 500 hab): 19€/month — basic alerts + reports
- **Village** (501–3.5k): 49€/month — + agenda + news
- **Bourg** (3.5k–10k): 99€/month — + messaging
- **Bastide** (10k–25k): 189€/month — + advanced reporting + analytics
- **Cité** (25k–50k): 390€/month — + custom branding + priority support
- **Métropole** (>50k): 590€/month — + white-label + dedicated account manager

### Target Tone
- Accessible to non-tech municipalities
- Professional but approachable
- Emphasizes citizen trust & civic engagement
- NOT corporate/sterile — should feel human
