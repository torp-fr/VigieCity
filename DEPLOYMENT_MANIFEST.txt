DATABASE SCHEMA CHANGES
===============================================================================

NEW TABLES:

moderation_queue:
  - id (UUID, PRIMARY KEY)
  - report_id (UUID, FOREIGN KEY)
  - collectivity_id (UUID, FOREIGN KEY)
  - status (ENUM)
  - auto_flagged (BOOLEAN)
  - confidence_score (NUMERIC)
  - flags (TEXT[])
  - moderator_notes (TEXT)
  - reviewed_by (UUID, FOREIGN KEY)
  - reviewed_at (TIMESTAMPTZ)
  - created_at (TIMESTAMPTZ)
  - updated_at (TIMESTAMPTZ)
  
  Indexes:
  - (status, collectivity_id, created_at DESC)
  - (report_id)


report_flags:
  - id (UUID, PRIMARY KEY)
  - report_id (UUID, FOREIGN KEY)
  - user_id (UUID, FOREIGN KEY)
  - reason (ENUM)
  - description (TEXT)
  - is_auto_flag (BOOLEAN)
  - created_at (TIMESTAMPTZ)
  
  Indexes:
  - (report_id, created_at DESC)


MODIFIED TABLES:

reports:
  - Added: moderation_status (ENUM, DEFAULT 'pending')
  - Added: auto_filtered (BOOLEAN, DEFAULT false)
  - Added Index: (moderation_status, created_at DESC)
